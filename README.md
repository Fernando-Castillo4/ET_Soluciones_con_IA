# ET_Soluciones_con_IA

## Descripcion

Este proyecto corresponde a la asignatura de Ingenieria de soluciones con Inteligencia Artificial, el proyecto integra tecnicas de IA para procesar
información y generar respuestas segun el prompt pre establecido, además de recibir restricciones y generarcion de 

## Tecnologias utilizadas

    Python 3.11.9: Lenguaje principal para el desarrollo

    Langchain: Frameworks para la construccion del 
               agente de IA

    OpenAI: Modelo utilizado GPT -4.1

    Jupyter Notebook: Entorno de desarrollo

    Pandas: Manipulacion de datos y almacenamiento

    Psutil: Obtencion de métricas

    Streamlit: Desarrollo de dashboard

    CSV: Almacenamiento de logs y metricas del chatbot

## Funcionalidades principales

- Procesamiento de consultas pre operatorias mediante IA
- Respuestas automáticas sobre el tema
- Manejo de conversaciones tipo chatbot
- Restricciones mediante filtros de palabras, prompt y conocimiento

## Instalacion y ejemplo de uso

Clona el repositorio:

git clone https://github.com/Fernando-Castillo4/ET_Soluciones_con_IA.git

Entra al proyecto y instala dependencias.

Para ejecutar el proyecto, debe agregar su token de github modelo gpt -4.1, luego iniciar todas las celdas de codigo una por una o presionando el boton "Run All", la ultima celda soltara un mensaje y estara listo para utilizar

Ejemplo de uso:
Usuario ¿Qué me recomiendas hacer antes de una operacion?
IA: Te recomiendo seguir estos procedimientos...
